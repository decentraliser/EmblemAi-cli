# Changelog

All notable changes to `@emblemvault/agentwallet` will be documented here.

## [3.3.0] - 2026-04-22

### Added
- Agent-autonomous token refresh via `tryAutoRefresh` — keeps long-running agent sessions authenticated without manual re-login.
- Encrypted secret storage (`safe`) for agent-managed credentials.
- Tempo MPP (Micro Payment Protocol) support, tempo-only mode toggle.
- Interactive model picker with OpenRouter search, live pricing, and session persistence.
- MCP server install section in README.

### Changed
- Dependencies refreshed (`@emblemvault/auth-sdk` ^2.3.18, `@x402/*` ^2.8.0, others).
- Default model set updated.
- Rename remaining "Hustle" references to "EmblemAI" across CLI copy.
- README GEO alignment: 200+ tools, dual audience framing, corrected GitHub link.
